
1. Why is wrapping logic in a function better than repeating the same if/else block?
A: Wrapping logic in a function is better than repeating the same if/else block because it makes the code more readable and reusable. 
It helps improve code readability and helps streamline the debugging and testing process.

2. Why is return often better than echo inside reusable functions?
A: Return is often better than echo inside reusable functions because it offers better flexibility and control for its usage such as
being able to pass to other functions. Using return also helps prevent unexpected output such as how echo immediately forces output.
